
# Wallet App

This is a wallet to store British coins and 1 pound notes.

## Wallet API

Can be used for
- Asking balance
- Adding money
- Removing money

# How to start the app

## Spin up infrastructure

```bash
docker compose up
```

## Run API

```bash
cd WalletApi
dotnet run
```
Open the swagger page and some money: http://localhost:5213/swagger/index.html
